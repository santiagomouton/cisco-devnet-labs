# Cisco DevNet Labs

Este repositorio recopila las practicas, ejercicios y laboratorios realizados durante los cursos de **Cisco Networking Academy (NetAcad)** orientados a **Cisco DevNet** y automatizacion de redes.

En el repositorio se incluyen ejemplos practicos enfocados en el uso de APIs, programacion con Python, automatizacion de dispositivos Cisco, consumo de servicios REST y otras herramientas utilizadas en el ecosistema DevNet. El objetivo principal es reforzar los conocimientos adquiridos en los cursos mediante ejercicios practicos y servir como material de consulta.
